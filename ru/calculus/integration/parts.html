# Интегрирование по частям

Метод интегрирования по частям – это один из основных приёмов интегрального исчисления наряду с взятием простейших интегралов и интегрировании с заменой переменной.

Какую задачу решает метод интегрирования по частям? Метод интегрирования по частям решает очень важную задачу, он позволяет интегрировать некоторые функции, отсутствующие в таблице, произведение функций, а в ряде случаев – и частное. В интегрировании нет удобной формулы наподобие (uv)′=u′v+v′u(uv)' = u'v + v'u(uv)′=u′v+v′u для вычисления производной произведения. Зато есть такая – формула интегрирования по частям собственной персоной

∫udv=uv−∫vdu\int udv = uv - \int vdu ∫udv=uv−∫vdu

По частям берутся интегралы следующих видов:

  1. ∫ln⁡(x)dx\int \ln(x)dx∫ln(x)dx, ∫(x2+3)ln⁡(x)dx\int (x^2 + 3)\ln(x)dx∫(x2+3)ln(x)dx, ∫xln⁡2(x)dx\int x\ln^2(x)dx∫xln2(x)dx - логарифм или логарифм, умноженный на какой-нибудь полином.

  2. ∫xexdx\int xe^x dx∫xexdx, ∫(x2−2x+5)e−2xdx\int (x^2 -2x +5)e^{-2x} dx∫(x2−2x+5)e−2xdx – экспоненциальная функция, умноженная на какой-нибудь полином. Сюда же можно отнести интегралы вроде ∫4xxdx\int 4^x x dx∫4xxdx – показательная функция, умноженная на полином, но в подавляющем большинстве случаев под интегралом находится еее.

  3. ∫xcos⁡6xdx\int x \cos 6x dx∫xcos6xdx, ∫(x2+3x)sin⁡2xdx\int (x^2 + 3x) \sin 2x dx∫(x2+3x)sin2xdx, ∫xtan⁡2xdx\int x \tan^2 x dx∫xtan2xdx – тригонометрические функции, умноженные на какой-нибудь полином.

  4. ∫arcsin⁡xdx\int \arcsin x dx∫arcsinxdx, ∫x2arctan⁡xdx\int x^2 \arctan x dx∫x2arctanxdx – обратные тригонометрические функции («арки»), умноженные на какой-нибудь полином.

Также по частям берутся некоторые дроби, соответствующие Заданиеы мы тоже подробно рассмотрим.

# Интегралы от логарифмов

Пример 1. Найти неопределенный интеграл.

∫ln⁡xdx\int \ln{x} dx ∫lnxdx

Воспользуемся формулой интегрирования по частям

∫udv=uv−∫vdu\int udv = uv - \int vdu ∫udv=uv−∫vdu

Формула применяется слева направо. Смотрим на левую часть: ∫udv\int udv∫udv. Очевидно, что в нашем Заданиее (и во всех остальных, которые мы рассмотрим) что-то нужно обозначить за uuu, а что-то – за dvdvdv.

В интегралах рассматриваемого типа за uuu всегда принимается логарифм.

Оформление решения реализуется следующим образом:

u=ln⁡xu = \ln xu=lnx dv=dxdv = dxdv=dx

То есть за uuu мы обозначили логарифм, а за dvdvdv – оставшуюся часть подынтегрального выражения.

Следующий этап – находим дифференциал dududu:

u=ln⁡x⇒du=(ln⁡x)′dx=1xdxu = \ln{x} \Rightarrow du = (\ln x)'dx = \dfrac{1}{x} \, dxu=lnx⇒du=(lnx)′dx=x1​dx dv=dxdv = dxdv=dx

Теперь находим функцию vvv. Для того чтобы её найти, необходимо проинтегрировать правую часть нижнего равенства dv=dxdv = dxdv=dx:

u=ln⁡x⇒du=(ln⁡x)′dx=1xdxu = \ln{x} \Rightarrow du = (\ln x)'dx = \dfrac{1}{x} \, dxu=lnx⇒du=(lnx)′dx=x1​dx dv=dx⇒v=∫dx=xdv = dx \Rightarrow v = \int dx = xdv=dx⇒v=∫dx=x

Теперь открываем наше решение и записываем правую часть формулы (uv−∫vduuv - \int vduuv−∫vdu)

∫ln⁡xdx=xln⁡x−∫x∗1xdx=xln⁡x−∫dx=xln⁡x−x+C\int \ln{x} dx = x \ln{x} - \int x * \dfrac{1}{x} \, dx = x \ln{x} - \int dx = x \ln{x} - x + C ∫lnxdx=xlnx−∫x∗x1​dx=xlnx−∫dx=xlnx−x+C

Где C=constC = constC=const.

В произведении uvuvuv я сразу переставил местами uuu и vvv, так как множитель xxx принято записывать перед логарифмом.

Как видно, применение формулы интегрирования по частям свело наше решение к двум простым интегралам.

Обратите внимание, что в ряде случаев сразу после применения формулы, под оставшимся интегралом обязательно проводится упрощение – в рассматриваемом Заданиее мы сократили подынтегральное выражение на xxx.

Выполним проверку. Для этого нужно взять производную от ответа:

(xln⁡x−x+C)′=(xln⁡x)′−(x)′+(C)′=(x)′ln⁡x+x(ln⁡x)′−1+0=(x \ln{x} - x + C)' = (x \ln x)' - (x)' + (C)' = (x)' \ln{x} + x(\ln x)' - 1 + 0 = (xlnx−x+C)′=(xlnx)′−(x)′+(C)′=(x)′lnx+x(lnx)′−1+0=

=1∗ln⁡x+x∗1x−1=ln⁡x+1−1=ln⁡x= 1 * \ln{x} + x * \dfrac{1}{x} - 1 = \ln{x} + 1 - 1 = \ln x =1∗lnx+x∗x1​−1=lnx+1−1=lnx

Получена исходная подынтегральная функция, значит, интеграл решён правильно.

В ходе проверки мы использовали правило дифференцирования произведения (uv)′=u′v+v′u(uv)' = u'v + v'u(uv)′=u′v+v′u, и это не случайно. Формула интегрирования по частям и формула нахождения производной от произведения – это два взаимно обратных правила.

Пример 2. Найти неопределенный интеграл.

∫xln⁡2xdx\int_{}^{}{x \ln^2{x} dx} ∫​xln2xdx

Подынтегральная функция представляет собой произведение логарифма на полином.

Как уже говорилось, за uuu необходимо обозначить логарифм (то, что он в степени, значения не имеет). За dvdvdv обозначаем оставшуюся часть подынтегрального выражения.

Записываем в столбик

u=ln⁡2xu = \ln^2{x}u=ln2x dv=xdxdv = xdxdv=xdx

Сначала находим дифференциал u=ln⁡2x⇒du=(ln⁡2x)′dx=2ln⁡x(ln⁡x)′dx=2ln⁡xxdxu = \ln^2{x} \Rightarrow du = (\ln^2{x})'dx = 2\ln{x}( \ln{x})'dx = \dfrac{2\ln{x}}{x} dxu=ln2x⇒du=(ln2x)′dx=2lnx(lnx)′dx=x2lnx​dx

Здесь использовано правило дифференцирования сложной функции

(u(v))′=u′(v)∗v′(u(v))' = u'(v) * v' (u(v))′=u′(v)∗v′

Теперь находим функцию vvv, для этого интегрируем правую часть второго равенства

dv=xdx⇒v=∫xdx=x22dv = xdx \Rightarrow v = \int{xdx} = \dfrac{x^2}{2}dv=xdx⇒v=∫xdx=2x2​

Для интегрирования мы применили простейшую табличную формулу

∫xndx=xn+1n+1+C\int{x^{n} dx} = \dfrac{x^{n + 1}}{n + 1} + C ∫xndx=n+1xn+1​+C

Теперь всё готово для применения формулы

∫xln⁡2xdx=x2ln⁡x2−∫x22∗2ln⁡xxdx=x2ln⁡x2−∫xln⁡xdx\int{x \ln^2{x} dx} = \dfrac{x^{2} \ln{x}}{2} - \int{\dfrac{x^{2}}{2} * \dfrac{2\ln{x}}{x} dx} = \dfrac{x^{2} \ln{x}}{2} - \int{x\ln{x} dx} ∫xln2xdx=2x2lnx​−∫2x2​∗x2lnx​dx=2x2lnx​−∫xlnxdx

Под интегралом у нас снова полином на логарифм. Поэтому решение опять прерывается и правило интегрирования по частям применяется второй раз. Не забываем, что за uuu в похожих ситуациях всегда обозначается логарифм.

u=ln⁡x⇒du=1xdxu = \ln{x} \Rightarrow du = \dfrac{1}{x} dxu=lnx⇒du=x1​dx

dv=xdx⇒v=x22dv = xdx \Rightarrow v = \dfrac{x^{2}}{2}dv=xdx⇒v=2x2​

Хорошо бы, если к данному моменту простейшие интегралы и производные Вы умели находить устно.

Возвращаемся к основному примеру и следим за знаками

x2ln⁡x2−∫xln⁡xdx=x2ln⁡x2−(x2ln⁡x2−∫x22∗1xdx)=x2ln⁡x2−x2ln⁡x2+12∫xdx=\dfrac{x^{2} \ln{x}}{2} - \int {x\ln{x} dx} = \dfrac{x^{2} \ln{x}}{2} - ( \dfrac{x^{2}\ln{x}}{2} - \int {\dfrac{x^{2}}{2} * \dfrac{1}{x} dx} ) = \dfrac{x^{2} \ln{x}}{2} - \dfrac{x^{2}\ln{x}}{2} + \dfrac{1}{2}\int \text{xdx} = 2x2lnx​−∫xlnxdx=2x2lnx​−(2x2lnx​−∫2x2​∗x1​dx)=2x2lnx​−2x2lnx​+21​∫xdx=

=x2ln⁡x2−x2ln⁡x2+12∗x22+C=x22(ln⁡x−ln⁡x+12)+C= \dfrac{x^{2} \ln{x}}{2} - \dfrac{x^{2}\ln{x}}{2} + \dfrac{1}{2} * \dfrac{x^{2}}{2} + C = \dfrac{x^{2}}{2}( \ln{x} - \ln{x} + \dfrac{1}{2} ) + C =2x2lnx​−2x2lnx​+21​∗2x2​+C=2x2​(lnx−lnx+21​)+C

Необходимость дважды (а то и трижды) применять правило интегрирования по частям возникает не так уж и редко.

# Интегралы от обратных тригонометрических функций

Общее правило: за uuu всегда обозначается обратная тригонометрическая функция.

Напоминаю, что к обратным тригонометрическим функциям относятся арксинус, арккосинус, арктангенс и арккотангенс. Для краткости записи я буду называть их «арками»

dv=dx⇒v=xdv = dx \Rightarrow v = xdv=dx⇒v=x

∫atan⁡2xdx=xatan⁡2x−∫2x1+4x2dx=xatan⁡2x−14∫d(1+4x2)1+4x2=\int {\operatorname{atan}{2x} dx} = x\operatorname{atan}{2x} - \int {\dfrac{2x}{1 + 4x^{2}} dx} = x\operatorname{atan}{2x} - \dfrac{1}{4}\int \dfrac{d( 1 + 4x^{2} )}{1 + 4x^{2}} = ∫atan2xdx=xatan2x−∫1+4x22x​dx=xatan2x−41​∫1+4x2d(1+4x2)​=

xatan⁡2x−14sin⁡2(1+4x2)+Cx\operatorname{atan}{2x} - \dfrac{1}{4}\sin^2( 1 + 4x^{2} ) + C xatan2x−41​sin2(1+4x2)+C

Интеграл найден методом подведения функции под знак дифференциала, можно использовать и метод замены в «классическом» виде.

Таким образом, помимо «чистого» интегрирования по частям нередко требуется применять и другие методы, приёмы решения.
